Application Web Mon site personnel

Projet Personnel — Developpement d'un site Web dynamique fullstack (React/Vite + PHP API + MariaDB + Apache2).

← Retour aux projets
Résumé

Application web auto‑hébergée sur un Raspberry Pi : le site affiche un bandeau d’annonces pour les nouveautés, génère un CV dynamique (contenu mis à jour facilement), et propose une gestion des utilisateurs (inscription/connexion, droits & accès). Une fois connecté, l’utilisateur peut accéder à des contenus réservés, notamment la mise à disposition des cours pour les inscrits.

Stack
Outils
  • Framework : React + Vite
  • IDE : VS Code
Étapes / Visuels
Frontend — React (Vite)
  • Navigation : application SPA avec routing (pages Accueil, CV, Cours, Espace membre).
  • Interface : composants réutilisables, design responsive (mobile/desktop) + UI claire.
  • Contenu dynamique : bandeau d’annonces + sections qui se mettent à jour sans recharger la page.
  • Authentification : connexion/inscription, affichage conditionnel selon droits & accès.
  • Accès membres : téléchargement/consultation des cours réservés aux inscrits.
Backend PHP
  • API PHP : endpoints REST (CRUD), échanges JSON, validation des données.
  • Sécurité : sessions/cookies, gestion des droits (accès public vs espace membre).
  • Base de données : MariaDB/MySQL, données persistantes (utilisateurs + contenus).
Base de données — MariaDB (phpMyAdmin)
  • Données persistantes : utilisateurs, annonces, cours et contenus du site.
  • Relationnel : tables + relations (IDs), requêtes SQL pour lire/écrire les infos.
  • Administration : gestion via phpMyAdmin (CRUD, import/export, sauvegardes).
+ +
Installation / Accès
  • Raspberry : Raspbian + SSH/VNC, installation Apache2 + sécurisation de base (firewall, ports).
  • Frontend : React (Vite) — interface web responsive, appels API et affichage dynamique des données.
  • Backend : PHP (API REST) — endpoints CRUD, validation des entrées, gestion des erreurs, échanges JSON.
  • Base de données : MariaDB/MySQL — schéma relationnel, tables + relations, requêtes SQL, données persistantes.
