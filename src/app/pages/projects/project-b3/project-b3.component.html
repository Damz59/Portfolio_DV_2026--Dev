Résumé
Contexte

Création d’une page d’accueil vitrine pour la chocolaterie fictive ChocoBoom afin de travailler la structure HTML, la mise en page CSS et le responsive.

Objectifs
  • Header + navigation (logo + menu 3 liens)
  • Contenu principal : section + aside (texte + visuel + blocs d’infos)
  • Responsive (desktop 2 colonnes → tablette en colonne → mobile sans aside)
  • Style & interactions : couleurs du modèle + transition au survol du menu

+
Étapes / Visuels
ChocoBoom — page d’accueil
  • But : site vitrine “ChocoBoom” (chocolaterie) — exercice responsive en HTML/CSS
  • Contenu : header + navigation (logo + 3 liens), contenu principal (section + aside)
  • Mise en page : Flexbox, alignements, espacements, centrage
  • Responsive : desktop (2 colonnes) → tablette (1 colonne) → mobile (aside masqué)
  • Stack : HTML5 + CSS3 (media queries + transitions)
HTML5 — structure & imports
  • Structure sémantique HTML5 : <header>, <nav>, <main>, <section>, <aside>
  • Meta viewport pour le responsive : <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Liaison des styles : <link rel="stylesheet" href="css/normalize.css"> + <link rel="stylesheet" href="css/style.css">
  • Intégration d’assets : images (<img ... alt="...">) + favicon
  • Navigation par liens : <a href="...">
  • Sélecteurs pour le CSS : class et id (ex : class="container", id="logo")
CSS — structure & imports
  • Mise en page responsive avec media queries (@media (max-width: 960px), @media (max-width: 560px))
  • Disposition avec Flexbox (display: flex, flex: 1, flex-direction, centrage)
  • Variables CSS pour les couleurs : :root + --bg-page, etc. + var(...)
  • Centrage / largeur de contenu : max-width + margin: 0 auto
  • Gestion des tailles avec box-sizing: border-box (sur *, *::before, *::after)
  • Typographie : font-family, font-size
  • Effets : transition sur le menu (transition: background-color 0.6s ease) + :hover
ChocoBoom
  • Flexbox : layout desktop 70/30 + nav en 3 “cases” (display: flex, flex:1, align-items, justify-content, gap)
  • Responsive : @media (max-width: 960px) (aside sous section), @media (max-width: 560px) (aside masqué)
  • Effets : hover + transition (menu : background-color 0.6s ease vers #a79189)
  • Organisation : variables CSS (:root) + normalize.css + classes réutilisées (.container)
