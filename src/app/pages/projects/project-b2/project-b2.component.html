Contexte

Objectifs
  • Header + navigation
  • Sections (mise en avant produits / catégories / infos)

Étapes / Visuels
ShoeSplash — page d’accueil
  • But : vitrine d’un magasin de chaussures (HTML/CSS)
  • Contenu : header + navigation, sections (produits / catégories / infos)
  • Mise en page : flexbox, alignements, espacements
  • Responsive : mobile → desktop
  • Stack : HTML5 + CSS3
HTML5 — structure & imports
  • HTML5 : balises sémantiques header, nav, main, section, article, footer
  • Liens CSS (dans le head) :
    <link rel="stylesheet" href="css/normalize.css"> (Normalize.css)
    <link rel="stylesheet" href="css/style.css"> (ta CSS)
  • Google Fonts : import de la police Courgette (fonts.googleapis.com / fonts.gstatic.com)
  • Font Awesome : import via link cdnjs, puis utilisation :
    <i class="fa fa-shopping-cart"></i>, <i class="fa fa-lock"></i>, etc.
  • Classes / IDs :
    classes : container, topbar, left, right, clear, escarpin
    ids : cart, slider, select, news, new, best, crush, info, bottombar
  • Images : img src="..." + alt (présent sur le logo, vide sur les autres images)
CSS — structure & imports
  • Flexbox : display: flex, align-items, justify-content, gap
  • Positionnement : position relative/absolute (ex : panier)
  • Responsive : @media (max-width: 576px)
  • Effets : hover, transitions (boutons, cartes, images)
  • Organisation : classes réutilisables + variables CSS
